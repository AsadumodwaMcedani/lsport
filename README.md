# LS Port — LBBS Student Portal

Private work-management system + student query portal. UFH lecturer project.

- **Start here:** `CLAUDE.md` (project memory), `memory/PROGRESS.md` (state), `docs/SPEC.md` (full spec)
- **Run locally:** `cd server && cp .env.example .env && npm i && npm run migrate && npm run seed && npm run dev` then `cd client && npm i && npm run dev` → http://localhost:5173
- **Tests:** `cd server && npm test`
- **Deploy:** see `docs/deployment/DEPLOY.md`
- Domains: work.lbbs.co.za (admin) · portal.lbbs.co.za (students/tutors)
